Próximas etapas técnicas

1. Registrar Login with AmazonCriar o Security Profile na Amazon e cadastrar a Redirect URI desta página.
2. Criar backend de tokensTrocar o authorization code por access token/refresh token fora do navegador.
3. Vincular Alexa SkillConfigurar Account Linking na skill para associar a conta Alexa à conta interna.
4. WhatsApp oficialPara automação real, usar WhatsApp Business Platform/API em vez de controlar WhatsApp Web embutido.
